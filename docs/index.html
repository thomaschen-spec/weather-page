Weather Page — 排程自動更新練習

這是什麼

一個純靜態的天氣頁面，重點不是天氣本身，是練「排程自動化」—— 每小時自動抓資料、自動產生新頁面、自動部署，全程不用人手動。

打開實際網頁 →

網址每 5 分鐘自動重新整理一次，永遠是最新天氣。

技術組合

GitHub ActionsOpen-Meteo APIVercel
  • 資料來源：Open-Meteo（免金鑰，公開氣象資料）
  • 排程：GitHub Actions 每小時整點過 5 分自動觸發一次
  • 部署：Vercel 偵測到新 commit 自動重新部署（純靜態，無需資料庫）

流程

GitHub Actions 排程 → 呼叫 Open-Meteo API 抓最新天氣 → 套進樣板產生新頁面 → git commit + push → Vercel 自動重新部署 → 打開網址就是最新一版。
